# Contributing to platformkit-community

This repository exists to make the platformkit community experience clear,
welcoming, and well moderated.

## Use this repository for

- Questions about the product and launch
- Roadmap feedback
- Announcement discussion
- Community process and moderation improvements

## Use the flagship repository for

- Product bugs
- Release-surface defects
- Public flagship contract changes

The flagship repository is
[septagon-dev/platformkit](https://github.com/septagon-dev/platformkit).

## Contribution standards

- Keep changes community-facing and lightweight.
- Improve routing, clarity, moderation, and support surfaces.
- Do not duplicate product implementation details here.
- Keep security-sensitive details out of public discussions.

## Good first contributions

- Improve support routing
- Tighten community documentation
- Clarify launch FAQs and announcements
- Turn repeated discussion themes into clear roadmap feedback

By participating in this repository, you agree to the standards in
[CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
