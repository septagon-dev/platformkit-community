# Support

## Where to go

- Questions, feedback, and roadmap conversation:
  this repository's Discussions tab
- Product bugs and flagship defects:
  [septagon-dev/platformkit issues](https://github.com/septagon-dev/platformkit/issues)
- Security reports:
  [SECURITY.md](SECURITY.md)

## Community support model

Support here is community and maintainer best effort. Use clear titles,
reproducible details, and the correct repository boundary to get useful help.
