# platformkit-community

The public discussion and community coordination repository for platformkit.

Use this repository for launch feedback, roadmap conversation, announcements,
questions, and community moderation. Product bugs and flagship contract changes
still belong in
[septagon-dev/platformkit](https://github.com/septagon-dev/platformkit).

## Start here

- Community guide: [CONTRIBUTING.md](CONTRIBUTING.md)
- Support paths: [SUPPORT.md](SUPPORT.md)
- Security reporting: [SECURITY.md](SECURITY.md)
- Community rules: [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)
- Repository scope: [REPO_CHARTER.md](REPO_CHARTER.md)
- Roadmap and launch sequence: [ROADMAP.md](ROADMAP.md)

## What this repository is for

- Discussion threads and launch feedback
- Announcements and roadmap conversation
- Community onboarding and moderation guidance
- Routing users to the right flagship or support surface

## What this repository is not for

- Product implementation work
- Private security reports
- Module-level bug tracking that belongs in an owning repository

## Community workflow

- Ask questions and share launch feedback here.
- Escalate product bugs to
  [septagon-dev/platformkit](https://github.com/septagon-dev/platformkit/issues).
- Keep security issues private and use the process in [SECURITY.md](SECURITY.md).

## Current status

This repository stays intentionally lightweight. Its job is to make the public
community experience clear and usable, not to duplicate product code or act as
a second issue tracker.
