# Security Policy

This repository is not the place for public security reports.

## How to report a vulnerability

- Do not post security issues in community discussions or public issues.
- Report vulnerabilities through the flagship repository process documented at
  [septagon-dev/platformkit/SECURITY.md](https://github.com/septagon-dev/platformkit/blob/main/SECURITY.md).
- If you need a private fallback, email `security@septagon.dev`.

## What belongs here

- Non-sensitive security documentation improvements
- Routing users toward the correct private reporting path

## What does not belong here

- Public exploit details
- Vulnerability disclosures
- Bug reports for product components
